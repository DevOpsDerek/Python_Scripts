# 🏋️ Follow-Along Exercises

Ten hands-on exercises to practise what you've learned from the script library.
Each exercise has a **starter file** (`exercise.py`) with `TODO` markers guiding you step by step,
and a **solution file** (`solution.py`) to check your work afterwards.

> **How to use these:**
> 1. Read the `exercise.py` file top to bottom — the comments explain everything.
> 2. Fill in the `TODO` sections yourself, one at a time.
> 3. Run your script frequently to test as you go: `python exercise.py`
> 4. Only peek at `solution.py` when you're truly stuck or want to compare approaches.

## Tips for Beginners

- **Don't panic at errors** — read the last line of the traceback first. That's the actual problem.
- **Use `print()` everywhere** while debugging. Remove them when it works.
- **Python docs**: https://docs.python.org/3/library/
- **Look up any function**: `help(sorted)` in the Python REPL gives you instant docs.
- **Google is allowed** — real developers search constantly. The skill is knowing *what* to search for.
